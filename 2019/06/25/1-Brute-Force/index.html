1. Brute Force

2019-06-25

Brute Force

가능한 경우의 수를 일일이 나열하면서 답을 찾는 방법 (완전 탐색)

재귀 호출

​ 범위가 작아질수록 유사한 형태를 반복하는 작업에 사용

​ 여러 조각으로 쪼갠 유사한 작업 중 한 조각을 수행하고 자기 자신을 다시 호출해 실행

​ 기저 사례 : ‘더 이상 쪼개지지 않는’ 최소한의 작업. 조건문을 통해 답을 곧장 반환

​ vs. 반복문 : n개의 원소를 자유롭게 사용 가능

접근법

  1. 입력에 따른 시간 복잡도 계산
  2. 가능한 모든 경우를 만드는 과정을 여러 개의 선택으로 나누기
  3. 하나의 조각을 선택 후, 나머지 답을 재귀 호출을 통해 완성
  4. 기저 사례 처리

문제

1. 소풍 (난이도:下)

내 풀이

​ 재귀 함수를 통한 완전 탐색

​ remain을 통해 짝의 주체와 짝을 찾을 때의 차이를 줌

​ 중복 방지를 위해 앞의 학생에게 먼저 짝을 지정

시간복잡도 : 9 * 7 * 5 * 3 * 1

2. 게임판 덮기 (난이도:下)

내 풀이

​ 마찬가지로 재귀 함수 사용

​ 중복 방지를 위해 더 위에 있고(i가 작은), 더 왼쪽에 있는(j가 작은) 흰 칸을 먼저 덮음

​ 이 때, 가능한 모양은 ┏, ┓, ┛, ┗ 이렇게 4개

문제점

​ 왼쪽 위를 선택함에 따라 모양이 위의 4개만 나온다는 것을 찾지 못했음

시간복잡도 : 4^16

+ 백준

  • 1065) 한수
  • 2309) 일곱 난쟁이
  • 2503) 숫자 야구

​ 치킨 배달, 구슬 탈출, 2048 풀어보자
