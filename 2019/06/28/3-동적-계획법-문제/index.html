3. 동적 계획법 문제

2019-06-28

1. 와일드카드(WILDCARD) (난이도:中)

내 풀이 과정
  • nowF와 nowW를 증가시켜가며 비교하는 완전 탐색 후 메모이제이션 적용하고자 함
문제점
  • now 값들을 인자로 갖는 재귀 함수를 생성하여 같을 때도 now+1을 인자로 재귀 호출을 하였는데 그러면 처음 ret 값도 같이 인자로 넘겨줬어야 함.
  • 각 비교 case 조건에 대한 이해도 부족으로 인한 구현 불가
  • 메모이제이션 적용법 생각 못 함
올바른 풀이
  • w와 f 값이 같거나 w가 ‘?’일 때는 while을 통해 간편히 비교, 한 함수 내에서 끝까지 비교 가능
  • 비교가 fail 됐을 때 원인에 대한 case 나눠줌
    1. nowW와 nowF가 배열에 끝까지 갔으면 두 문자열은 일치하므로 return true;
    2. w[nowW]가 ‘*’일 때는 *의 길이를 0 ~ 남은length 로 설정 후, 재귀 호출
  • 배열의 위치 now를 통해 memoization 적용
※ 완전 탐색 구현 방법에 대한 이해도 높이기 + 어떻게 메모이제이션을 적용시킬 것인가?
