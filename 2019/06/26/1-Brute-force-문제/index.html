3. 시계 맞추기 (난이도:中)

내 풀이 과정
  • 시계가 4번 돌면 원상태로 돌아오기 때문에 각 시계는 최대 3번, 즉 0~3번만 돌음
  • 가장 적게 스위치를 누르고자 하기 때문에 누르는 스위치의 개수를 0개에서부터 증가시킴
  • 최대로 스위치를 누를 수 있는 개수는 30개. 30C0 ~ 30C30까지 확인해가며 해답 발견 시 중지
  • 시간복잡도 : 최악의 경우 2^30

문제점
  • 가장 적은 스위치를 고르는 최적화 문제여서 선택 스위치의 개수를 증가시켜가며 풀어야 한다고 생각하여 간단하게 각 스위치를 3개로 복제시켜 문제 발생
  • 30Cn을 하는 경우, 각각의 스위치가 3개씩 들어가 있음. 이로 인한 중복 케이스 발생으로 시간 초과

수정 사항
  • 10개의 시계가 각각 4가지 경우를 가지므로 총 탐색은 4^10(=2^20)번
  • 총 탐색의 수가 많지 않으므로 모든 경우 탐색 후 최솟값을 찾아주면 됨
