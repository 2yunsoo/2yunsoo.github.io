분할 정복 (Divide & Conquer)

주어진 문제를 둘 이상의 부분 문제로 나눈 뒤 각 문제에 대한 답으로부터 전체 문제의 답을 계산

  1. divide : 문제를 더 작은 문제로 분할
  2. merge : 각 문제의 답을 원래 문제의 답으로 병합
  3. base case : 기저 사례

재귀호출 vs 분할정복

​ 한 조각 vs 나머지 전체 거의 같은 크기의 부분 문제들

예제

1. 수열의 합

재귀호출의 경우 n번의 함수 호출이 필요하나 분할 정복의 경우 함수가 호출될 때마다 두 번에 한 번은 n이 절반으로 줄어든다. 따라서 실행 시간은 O(log n)가 된다는 것을 알 수 있다.

2. 행렬의 거듭제곱

A^m = A^(m/2) × A^(m/2) if(m%2==0)

A^m = A × A^(m-1) if(m%2==1)

3. Merge Sort & Quick Sort

병합 정렬 : 배열을 가운데에서 쪼개 비슷한 크기의 수열 두 개로 만드는 과정을 반복 후 재귀 호출을 이용해 각각 정렬. 정렬된 배열을 하나로 합침.

시간복잡도 : 단계 수(O(log n)) × 한 단계 당 병합 시간 (O(n)) = O(n·logn)

퀵 정렬 : 배열을 단순하게 가운데에서 쪼개는 대신 pivot을 지정하여 작거나 같은 숫자는 왼쪽, 큰 숫자는 오른쪽으로 나누는 partition을 통해 병합 과정이 필요 없도록 분할.

시간복잡도 : pivot에 따라 달라짐. 평균 O(n·logn) / 최악 O(n^2)
