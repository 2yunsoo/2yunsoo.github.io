음주 코딩 (백준)

곱했을 때, 음수의 개수에 따라 부호가 달라지므로 짝수는 0, 홀수는 1로 바꿔 더해주면 된다. 0의 경우에는 최대 노드의 개수 10만보다 큰 값으로 더해주거나 따로 vector를 만들어서 체크해주면 된다. 큰 값을 더해주면 long long을 쓰는데다 안 이뻐지는 것 같길래 zeros를 만들어 체크해줬는데 중간에 구간트리랑 헷갈려서 논리가 꼬였다. 그래서 check vector를 하나 더 만들어서 그냥 처리해버렸다. 원래는 zeros(pos) - zeros(pos-1) 해야되지 않나 싶다. 결론적으로는 코드도 길어지고 벡터도 두 개나 더 쓰게 된 것 같다.

EOF를 근 10년만에 다시 봤다. c = getchar(); if(c==EOF) return; 어릴 때는 이런 식으로 썼던 것 같은데 이렇게 말고 while(~scanf(“%d”,d)) ; 이렇게 쓰면 깔끔하게 해결된다.

scanf를 사용할 때, “%d “,&a와 “%d”,&a 에는 차이가 있다. 헷갈린다. 실수의 입력 때는 상관 없는 것 같은데 문자 입력을 받을 때 문제가 생겨 잘 해결봐야한다.
