너비 우선 탐색(Brute First Search, BFS)

시작점에서 가까운 정점부터 순서대로 방문하는 그래프 탐색 알고리즘

너비 우선 탐색에서 새 정점을 발견하는 데 사용한 간선들만을 모은 트리 : BFS 스패닝 트리

시간복잡도 - 인접리스트 : O(|V|+|E|) / 인접행렬 : O(|V|^2)

너비 우선 탐색, BFS는 최단 경로를 구하는 문제에 사용

간선 (u,v)를 통해 정점 v를 처음 발견했다면 v까지의 최단거리는 u까지의 최단거리 + 1

스패닝 트리에서의 경로 = 그래프 상에서의 최단 경로
