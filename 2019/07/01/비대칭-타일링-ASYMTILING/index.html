비대칭 타일링(ASYMTILING)

2019-07-01
다이나믹, 메모이제이션, 하

비대칭 타일링(ASYMTILING) (난이도:下)

내 풀이

  • ret = (tile(width-1) + tile(width-2)) % MOD로 너비 n의 전체 타일링 방법 수를 구함

  • 대칭인 타일링 방법은 tile(n/2) + tile(n/2-1) if(n%2==0)

    ​ tile((n-1)/2) if(n%2==1)

문제점
  • 음수의 mod 연산은 불가능
올바른 풀이
  • 음수가 나올 가능성이 있으니 출력 시 MOD를 한 번 더해주고 다시 mod 연산 취해줌
  • 짝수의 경우 두 경우를 빼기 때문에 2*MOD를 더해줌
느낀 점
  • 음수라고 mod도 못하다니… 화난다……
