단어 제한 끝말잇기(WORDCHAIN)

2019-07-08
DFS

단어 제한 끝말잇기(WORDCHAIN) (난이도:下)

내 풀이
  • 각 단어의 시작과 끝의 알파벳을 정점으로 각 단어를 간선으로 갖는 그래프를 생성
  • 각 정점마다 들어오는 간선과 나가는 간선의 개수를 확인
    • 모두 짝수라면 오일러 서킷
    • 나가는 간선의 개수가 하나 더 많은 정점(start), 들어오는 간선의 개수가 하나 더 많은 정점(end)이 각각 한 개씩 존재하면 오일러 트레일
    • 이 외에는 모두 IMPOSSIBLE
  • 그래프가 오일러 서킷이라면 dfs() 함수를 통해 출력
  • 그래프가 오릴러 트레일이라면 간선 (end,start)를 추가해주고 end를 시작지점으로 dfs() 함수 수행
  • 처음에는 무조건 end -> start로 가게 만들고 나중에 출력 시 첫 번째는 제외
문제점

ㅅㅄㅄㅄㅄㅄㅄㅄㅄㅂ 진짜 죽어도모르겠다 진짜… 사실 짜잘한 변수 설정 같은 문제가 계속 있긴 했는데,,, 진짜 겁나 고치고 고쳐댔는데 안풀림 논리는 저거 맞는 것 같은데… ㅅㅂㅅㅂㅅㅂ 얘도 늦어도 수요일까지…
