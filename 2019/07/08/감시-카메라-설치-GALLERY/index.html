감시 카메라 설치(GALLERY) (난이도:中)

내 풀이
  • ‘한 번 관람한 갤러리를 다시 가기 위해서는 이전에 지나왔던 복도를 반드시 지나야 하는 구조’ : 사이클이 존재하지 않는 그래프
  • 연결된 간선이 1개인 정점에서부터 진행하는데…
문제점
  • 문제 이해를 잘못 했다…
  • 감시 카메라는 복도로 직접 연결된 갤러리들을 감시 : 바로 옆에 정점만 확인 가능…
올바른 풀이
  • 루트 없는 트리 - 사이클이 존재하지 않는 그래프로 노드 간의 상하 관계가 없는 트리와 같은 형태 띰
  • 아래 조건들은 모두 동치로 하나라도 성립하면 루트 없는 트리라고 판단 가능
    • V-1개의 간선을 가짐
    • 사이클이 존재하지 않음
    • 두 정점 사이를 연결하는 단순 경로가 정확히 하나 존재
  • 리프 노드는 자기 자신과 부모 노드밖에 연결되지 않지만 부모 노드는 위아래 둘 다 연결이 되어있음
  • 따라서 리프 노드 대신 그 부모 노드를 대신 선택
구현법
  1. 리프 노드는 선택하지 않음
  2. 트리에 맨 밑에서부터 올라오면서 자손 중 아직 감시 카메라와 연결이 안 된 노드가 있을 경우에만 현재 노드 선택
