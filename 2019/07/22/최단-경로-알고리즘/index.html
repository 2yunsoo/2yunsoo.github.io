최단 경로 알고리즘

음수 사이클이 존재하면 안됨
무방향 그래프는 두 개로 쪼개서 각각 방향 그래프로 바꿔줌

다익스트라(Dijkstra)

시작점에서부터 인접한 정점을 검사. 간선 (u,v)를 검사하였는데 아직 v를 방문하지 않았으면 u까지의 최단거리에 (u,v)의 가중치를 더한다. 이 값이 현재 저장되어 있는 dist[v]보다 작다면 갱신 후에 (dist[v],v)를 우선순위 큐에 넣는다.

우선순위 큐는 가장 큰 값을 자동으로 찾아서 빼 주기 때문에 최단 경로에 마이너스를 붙여 큐에 저장해주면 손쉽게 사용 가능.

구현
시간 복잡도

간선 검사 : |E|개 간선이므로 O(|E|)
우선순위 큐에 원소 삽입 : 최대 |E|개이므로 O(|E|log|E|)
합치면 O(|E|log|E|)이긴 한데 |E|는 아무리 커도 |V|^2보다는 작으므로 O(|E|log|V|)라고 해주자.

우선순위 큐를 사용하지 않는 다익스트라

정점의 수가 적거나 간선의 수가 매우 많은 경우에는 그냥 반복문을 사용하는게 더 빠를 경우도 있다. 여기서는 가 정점을 방문했는지 여부를 나타내는 배열이 하나 필요하게 된다. 시간복잡도는 O(|V|^2+|E|)
